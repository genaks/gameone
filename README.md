Memory Card Match Game
A classic memory card matching game built with Unity 2021.3.45f2.

**Getting Started**

**Opening the Project**

1. Open the project in Unity
2. Load the BootstrapScene to start the game
3. Press Play to begin
  
**How to Play**

Match pairs of cards by flipping them over two at a time. Find all matching pairs to win the game!

**Project Status**

This is a work-in-progress project. The core gameplay is functional, but some features are still being refined.
