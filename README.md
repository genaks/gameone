Memory Card Match Game
A classic memory card matching game built with Unity 2021.3.45f2.

**Getting Started**

**Opening the Project**

1. Open the project in Unity
2. Load the BootstrapScene to start the game
3. Press Play to begin
  
**How to Play**

Match pairs of cards by flipping them over two at a time. Find all matching pairs to win the game!

**How to add more levels**

1. Right click anywhere inside it's empty the project window
2. From the menu, choose Create->Data->Level
3. This will create a new scriptable object where one can specify the data about the level. Add this created object to the Level_Test scriptable object and it will be added to the current pool of levels being used inside the game.
